# Changelog

All notable changes to this project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [Unreleased]

### Added
- Initial project scaffolding: license, contribution guides, CI, and documentation set.
- Core design documents: Game Design, Technical Design, UX Specification, Art Direction,
  and Level Format.
- Deterministic slide-and-redirect simulation engine.
- BFS/A* solver, difficulty estimator, and procedural level generator.
- Canvas rendering pipeline, input handling, and synthesized audio.
- Full UI shell with campaign, level select, statistics, achievements, and settings.
- Level editor with validation, instant play, and share-codes.
- Handcrafted level campaign plus Daily and Endless modes.
- Progressive Web App support for offline play.
